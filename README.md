# 🎵 Audio Emotion Recognition using Machine Learning

This project classifies human emotions from speech audio using MFCC features and a CNN model.

## Emotions
Happy, Sad, Angry, Fearful, Neutral, Disgusted

## Tech Stack
Python, TensorFlow, Librosa, NumPy, Scikit-learn

## Datasets
RAVDESS, TESS, CREMA-D (not included due to size)

Team Lead – UG Major Project (BCA)
